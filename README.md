# Shopping_system_LVGL8
**一个 HUST C课设 课程项目。**  

**概述**  
>   模拟收银机软件，可以选择商品添加至结算列表，支持输入数量或重量。

**功能**

> 1.	预设至少5种商品及单价；  
> 2.	用户选择商品并输入重量信息；  
> 3.	计算总价并显示明细；  
> 4.	支持优惠活动（满减、打折等）；  
> 5.	显示购物清单；  
> 6.	设置多种优惠；  
> 7.	保存交易记录等。

**界面**

> 1.	清晰菜单导航；  
> 2.	商品列表展示；  
> 3.	购物车内容展示；  
> 4.	价格计算结果展示。  

本项目运行在 HUST AIA 专用开发板上。
